# Healthcare Enrollment & Claims Analytics

## Project Overview
This project analyzes healthcare enrollment and claims data to understand
membership trends, cost drivers, and plan performance. The goal is to design
clear business KPIs and build analytics-ready datasets that support
decision-making by leadership teams.

## Business Objectives
- Track active members and enrollment growth
- Monitor claims cost trends
- Identify high-cost members
- Compare performance across insurance plans

## Data Sources
- Members (enrollment and demographics)
- Claims (medical costs and utilization)
- Plans (plan metadata)

> Note: All data used in this project is synthetic and created for educational purposes only.

## Tools Used
- SQL (for data analysis and modeling)
- Power BI / Tableau (for visualization)
- GitHub (for version control and documentation)

## Project Status
This project is currently in progress. Data modeling, KPI calculations,
and dashboards will be added incrementally.
